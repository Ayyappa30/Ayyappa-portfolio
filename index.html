Project Description

Scanster is a comprehensive document scanning application developed using React Native for Android and iOS users. It offers advanced features, including local image and video uploads, PDF conversion, QR code generation, and timestamped image capture. The application efficiently utilizes RNFS and React Native Async Storage for data storage and integrates Google Authentication for secure access. Google Drive storage is employed for seamless data management, ensuring a user- friendly interface with efficient data storage and retrieval.
